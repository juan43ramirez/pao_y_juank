# Paola & Juan Camilo — Wedding Site

Static single-page wedding site (bilingual ES/EN), hosted on GitHub Pages.

- **Live:** https://juan43ramirez.github.io/pao_y_juank/
- **Stack:** plain HTML/CSS/JS, everything inline in `index.html`. No build step.
- To publish changes: commit and push to `main` — Pages redeploys in ~1 minute.
- RSVP backend: see [`apps-script/Code.gs`](apps-script/Code.gs) (setup steps in its header comment).
